跨域标准解决方案

fetch(ajax)接口的跨域解决方案 — 跨源资源共享(cors)

优点：解决异步接口请求过程中的各种跨域问题, 替代老的jsonp方案或者iframe跨域方案，从而带来更好的用户体验与开发体验
原理：就是使用自定义的HTTP头部让浏览器与服务器进行沟通，从而决定请求或响应是应该成功，还是应该失败

web客户端

  • 正常的通过fetch或者ajax请求数据，不需要额外设置

  • 如果请求需要带上cookie,需要请求头带上：withCredentials

  • 未避免cors出现先导请求(诸如：options请求)，导致发两次请求，降低性能；content-Type必须为application/x-www-form-urlencoded, multipart/form-data, 或text/plain中的一种

服务端

  • 服务端响应头中必须包含：

  • 如果请求带了cookie，且服务端允许跨站访问携带cookie，响应头必须包含

  • 如果允许web客户端读取服务器的头信息,需要设置Access-Control-Expose-Headers的白名单

iframe 跨域问题 - postMessage方案

  • 父页面发出message
  • iframe页面接收message

form异步提交的跨域方案：formdata + cors方案

JSONP

JSONP包含两部分：回调函数和数据。
回调函数的参数就是我们要获取的

web sockets

客户端和服务端的双向通信的协议
只有在支持web socket协议的服务器上才能正常工作。

css经典布局

发表于 2015-03-21 | 分类于 css |

经典布局

  1. 弹性盒子布局

    • 要素
      • display:flex
      • flex-direction: row | row-reverse | column | column-reverse
      • flex-wrap:wrap| nowrap| wrap-reverse 换行方式
      • align-items: flex-start | flex-end | center | baseline | stretch
      • justify-content: flex-start | flex-end | center | space-between | space-around
      • flex: auto | initial | none | inherit | [ flex-grow ] || [ flex-shrink ] || [ flex-basis ] flex 属性用于指定弹性子元素如何分配空间。

  2. 圣杯布局

    • 左中右布局，左边和右边固定宽度，中间宽度自适应,需要用到浮动，负边距，相对定位（相对于自身定位relative）

  3. 双飞翼布局

    • 左中右布局，左边和右边固定宽度，中间宽度自适应,需要用到浮动，负边距

  4. 头尾固定高度中间高度自适应布局

    • 兼容ie6的做法利用boxsizing改变盒子模型IE6/IE7下，的box-sizing默认值本就是border-box（注：IE7有一点点不正常，overflow:hidden后神志有所恢复）。
    • 不兼容ie6做法position

  5. 两列布局，左边固定，右边自适应的三种方法

    • 使用margin负值，使右边容器与左边同一行，并100%宽。再在右边容器中放一个子容器，设置margin-left值为左边容器的宽度。原理和双飞翼布局一样
    • 使用position，右边容器必须设置right

  6. 栅格布局：在一个有限的、固定的平面上，用水平线和垂直线（虚拟的线，“参考线”），将平面划分成有规律的一系列“格子”（虚拟的格子），并依托这些格子、或以格子的边线为基准线，来进行有规律的版面布局。

    • 原理：Flowline的宽度为W，column的宽度为c，Gutter的宽度为g，Margin的宽度为m，栅格列数为N

    　　W = cN + g(N-1) + 2m；g的宽度通常为m的两倍，所以：

    　　W = (c+g) * N；把c+g记为C，得：

    　　W = C * N;

    　　大部分的栅格系统都是此公式的变体。

    • Bootstrap的栅格系统,12列

    • Grid 100切分 百分比布局

every():

对数组中的每一项运行给定函数。如果函数对每一项都返回true，则返回true。

some():

对数组中每一项运行给定函数。如果函数对 任一项返回true，则返回true

filter():

对数组中的每一项运行给定函数。返回该函数会返回true的项组成的数组。

forEach():

对数组中每一项运行给定函数。该函数没有返回值。

map():

对数组中每一项运行给定函数。返回每次函数调用的结果组成的函数。
