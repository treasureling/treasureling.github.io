react生命周期小结

发表于 2017-02-23 | 分类于 react |

组件的生命周期分成三个状态：

Mounting：已插入真实 DOM

  • componentWillMount()

    服务器端和客户端都只调用一次，在初始化渲染执行之前立刻调用。

  • componentDidMount()

    在初始化渲染执行之后立刻调用一次，仅客户端有效（服务器端不会调用）。
    此时可以通过 this.getDOMNode() 来获取相应 DOM 节点。

Updating：正在被重新渲染

  • componentWillUpdate(object nextProps, object nextState)

    在接收到新的props 或者 state 之前立刻调用。在初始化渲染的时候该方法不会被调用。

  • componentWillReceiveProps

    可以作为react 在 prop 传入之后， render() 渲染之前更新 state 的机会

  • shouldComponentUpdate

    在接收到新的props 或者 state，将要渲染之前调用。该方法在初始化渲染的时候不会调用，在使用 forceUpdate 方法的时候也不会。

    • componentDidUpdate(object prevProps, object prevState)

      在组件的更新已经同步到DOM 中之后立刻被调用。该方法不会在初始化渲染的时候调用。
      使用该方法可以在组件更新之后操作DOM 元素。

Unmounting：已移出真实 DOM

  • componentWillUnmount()
    在组件从DOM 中移除的时候立刻被调用。
